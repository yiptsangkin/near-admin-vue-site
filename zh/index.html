near-admin-vue

基于vue和ant design的好用中后台前端框架

开始 →

多页多路由

支持多页多路由的模式，允许单独打包，合并打包，灵活度高

人性化功能

支持全局快捷键，全局字典，本地mock，动态路由，权限验证等功能

无需配置路由

开发过程中，如果没有特殊需求无需配置任何路由，方便省心

国际化

自带Google机翻功能，配置一个文件，轻松国际化

丰富组件

框架自带常用的业务组件，方便用户快速实现界面

升级方便

框架设计上已经考虑到后期升级，减少升级修改成本

# 开始

# clone the project from github
git clone https://github.com/yiptsangkin/near-admin-vue.git

# enter the project directory
cd near-admin-vue

# install dependency
npm install

# run application with all pages
npm run serve

# run application with single page and pick target entry
npmr run single:serve

? Pick target entry ›  
◯   auth
◯   manage

# 构建

# build application with all pages
# uat
npm run uat:build
# dev
npm run dev:build
# prod
npm run prod:build

# build application with single page
npm run single:build

? Pick target entry ›  
◯   auth
◯   manage

    no
