# 跨域问题

基于作者的开发经验，统一都使用代理的方式去处理跨域请求，在本地开发环境可以通过在vue.config.js中的devServer进行代理配置，在生产环境则可以使用nginx 进行代理配置。或者后端应用在程序上控制cors也可以，方法都比较简单。
