# 国际化

near-admin提供了非常便捷的国际化功能，首先他自带机翻脚本，开发者只需要维护一个文本json就可以通过google翻译多国语言，并且框架会自动处理引入功能， 方便快捷，同时支持runtime下的i18n开关功能，关闭国际化功能轻而易举。

# 翻译脚本

# translate
npm run translate

# 使用

near-admin是基于vue-i18n实现的国际化，因此使用方法也是一样的通过$t来实现不同语言下面内容的展示。

# 移除

在前面提到的个性化设置页面，near-admin提供了一个国际化的开关，可以实现国际化的移除，其核心原理是重写了$t的方法来兼容框架内置的一些国际化语言。 开发者如果不想要使用国际化功能，那么只需要把国际化开关关了，同时直接在代码中写入文本（不经过$t）即可。
