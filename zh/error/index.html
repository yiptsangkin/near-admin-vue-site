# 前端日志

near-admin基于Logline实现了前端日志的相关内容，并且在个人中心增加了前端日志的入口，同时也在request请求库里面提前封装好了相关的错误日志记录埋点， 因此，如果是一些接口问题或者是请求造成的脚本错误问题都可以自动捕捉到前端日志中，同时前端日志支持按日期查询和导出。
