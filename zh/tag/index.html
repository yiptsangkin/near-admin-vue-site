# 标签栏

标签栏部分主要提供的是页面组件的一个选择功能，开发者可以通过点击菜单或者框架提供的api新增、更新、关闭标签。标签对应访问的页面就是组件的路径，使用 起比较方便。

提示

标签列表对应vuex中的curTagList，当前选中的标签对应vuex中的curTagIndex。

# 标签栏右键

在标签上使用右键，具有有一定的右键操作功能，框架默认提供的功能有关闭当前标签、关闭右侧标签、关闭全部标签、关闭其他标签、刷新页面、独立页面。

提示

标签右键的菜单在NTagContentMenu中维护，虽然不建议修改core中的内容，但如果你有特殊的需求，可以通过修改这个组件来个性化定制你的右键功能。

# 关闭标签功能

默认提供了关闭当前标签、关闭右侧标签、关闭全部标签以及关闭其他标签的功能，主要通过官方提供的$closepage来实现。如果你希望通过api关闭某个标签， 需要传入两个参数，分别为要关闭的标签下标以及关闭类型，如果关闭标签有目标的跳转标签的话，可以通过target字段来实现。

export interface ClosePageOpt {
    // close page type
    // 1. 'cur': close current tag;
    // 2. 'right': close right tag;
    // 3. 'other': close other tag;
    // 4. 'all': close all tag
    type?: string,
    // current tag index
    idx: number,
    // after close tag, where it will go, enter the target index
    target?: number
}

# 独立页面功能

右侧菜单还提供了独立页面功能，对于webview的独立页面将会直接跳转到对应的网站，如果是内部的管理页面的话，将会动态添加缓存到路由中然后新开一个路由标签页。

# 固定标签功能

右键还提供固定标签功能，避免在关闭标签的时候不小心关闭未保存的标签页。
